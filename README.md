# cs41 - Crowd Source FAQs

Community-driven Q&A platform for internship seekers.

## Features

- Ask and browse FAQs
- Community-driven answers
- AI-powered FAQ assistance
- Search and recommendations
- Insights and leaderboard
- Authentication support

## Tech Stack

- React
- Vite
- Tailwind CSS
- Node.js
- Express
- SQLite

## Project Structure

```text
client/     React frontend
server/     Node.js backend
research/   Research files
```

## Setup

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

Frontend:
http://localhost:5173

Backend:
http://localhost:3001
