# Crowd Source FAQs

Community-driven Q&A platform for internship seekers.

## Features

- Ask and browse FAQs
- Community-driven question and answer platform
- AI-powered FAQ assistance
- Search and FAQ recommendations
- Insights and leaderboard pages
- React + Vite frontend
- Node.js backend with database support

## Project Structure

```text
client/     -> React frontend
server/     -> Node.js backend
research/   -> Research files and datasets
```

## Quick Start

### Backend

```bash
cd server
npm install
npm run seed
npm run dev
```

Backend runs on:

```text
http://localhost:3001
```

### Frontend

```bash
cd client
npm install
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```

Open:

```text
http://localhost:5173
```
