# Crowd Source FAQs

Community-driven Q&A platform for internship seekers.

## Quick Start

```bash
# Backend
cd server
npm install
npm run seed   # seed demo data
npm run dev    # starts on :3001

# Frontend (in another terminal)
cd client
npm install
npm run dev    # starts on :5173
```

Open http://localhost:5173
